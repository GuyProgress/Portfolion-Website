# Portfolio Website

A modern, responsive portfolio website showcasing career milestones and professional achievements.

## Features

- **Hero Section**: Welcoming introduction with call-to-action
- **Career Milestones Timeline**: Interactive timeline displaying key career achievements
- **Skills Section**: Overview of technical skills and expertise
- **Contact Section**: Easy ways to get in touch
- **Responsive Design**: Fully responsive, works on all devices
- **Smooth Animations**: Engaging animations and transitions

## Structure

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with responsive design
- `script.js` - JavaScript for interactive features

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content with your own information:
   - Update career milestones in the timeline section
   - Add your skills and expertise
   - Update contact information with your email and social links

## Customization

Edit the following sections in `index.html`:
- Hero section: Update title and subtitle
- Career Milestones: Add/modify milestone entries with your achievements
- Skills: Update skill cards with your expertise
- Contact: Update email and social media links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Animations)
- JavaScript (ES6+)
- Responsive Design
- Intersection Observer API
